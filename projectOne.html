Prework

  • Download Unity. (Make sure that AR support is selected when you install the software.)
  • Download any 3D model you want to use for this project.
  • Have an image ready (digitally and printed) to use as your image target.

Add a Vuforia Camera

Augmented reality experiences created using Vuforia uses a specific AR camera object. It replaces the default Main Camera object.

.
  • Go to GameObject -> Vuforia -> AR Camera
  • Delete the default Main Camera

Setting Up the Camera

  • In the Inspector Window, click "Open Vuforia Configuration"

The camera requires an app license key from Vuforia to ...

To get an App License Key:
  • Go to developer.vuforia.com
  • Create an account with Vuforia
  • Click the Develop Tab → “License Manager” --> Create a License Key
  • Copy your app license key that was generated and paste it into the box in the Inspector Window

Create database with our image targets

Image targets will be what your camera will be detecting to trigger the experience. The camera can be directed at a digital version of the image on a screen or a physical printed version.

  • Go to GameObject -> Vuforia -> Image
  • Swap out the default image with your own image

Adding in your 3D model

When the camera detect the image, it will trigger the experience, or 3D model to appear scaled and positioned relative to your image target.

  • Import your 3D model by going to
    GameObject -> 3DObject
  • In the Hierarchy Panel, drag your 3D model on top of the Image Target to make it a child of the image

Now your 3D model and image are connected and the program knows to trigger a specific 3D model when it detects a unique image.

Enable Vuforia Augmented Reality

Make sure that the Vuforia Augmented Reality option is selected in the player settings

  • Navigate to the Player Settings by going to File -> Build Settings -> Player Settings
  • Navigate to the bottom of the Inspector and expand XR settings
  • Check the box nect to Vuforia Augmented Reality
That's it! You're all done! Go and test your experience!
